NormalizedCollection

A library for joining paths and normalizing data.

Basics

A normalized collection wraps Firebase references and provides the same API. It can be used almost anywhere a regular Firebase reference would be appropriate, including libraries like AngularFire. 
// create a Firebase reference
var fb = new Firebase('https://fbutil.firebaseio.com/examples/profiles');

// map the paths we are going to join
var norm = new Firebase.util.NormalizedCollection(
  fb.child('login'),
  fb.child('profile')
);

// specify the fields for each path
norm = norm.select( 'login.username', 'profile.first', 'profile.last' );

// get a reference we can use like a normal Firebase instance
ref = norm.ref();

// run it and see what we get
ref.on('value', function(snap) {
  console.log('profile updated!', snap.name(), snap.val());
});
