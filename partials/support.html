Helpful tips to isolate a problem

  1. Start by reading the limitations and best practices
  2. Turn on debugging by calling Firebase.util.logLevel(true); (use 'info', 'log' or 'debug' to tweak verbosity)
  3. Check your JavaScript console for error messages
  4. Add a keyMap to every path and see if the problem resolves

The logging utility accepts several options and has grep capabilities. Read more at GitHub.
