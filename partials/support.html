Getting Help

Troubleshooting

  1. Turn on debugging by calling Firebase.util.logLevel(true); (use 'info', 'log' or 'debug' to tweak verbosity)
  2. Check your JavaScript console for error messages
  3. Double-check the API methods for any tips or limitations

The logging utility accepts several options and has grep capabilities. Read more at GitHub.
