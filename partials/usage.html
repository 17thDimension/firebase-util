Getting Started

On the Web

<script src="http://cdn.firebase.com/v0/firebase.js"></script>
<script src="https://cdn.firebase.com/libs/firebase-util/0.1.0/firebase-util.min.js"></script>
<script>
    var ref = Firebase.util.join( new Firebase(PATH1), new Firebase(PATH2), ... );
</script>

Node.js

Just manually include the js file in your node scripts. 
var Firebase = require('firebase');
var FirebaseUtil = require('./firebase-util.js');
var ref = FirebaseUtil.join( new Firebase(PATH1), new Firebase(PATH2), ... );

Full Documentation

For now, the README on GitHub serves as the full documentation for Firebase.util.join libs. It covers everything from dynamic path keys to best practices and limitations. Be sure to check it out after the examples.
